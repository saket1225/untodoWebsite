What is untodo?

untodo is a task management + accountability app. It tracks your daily tasks, monitors your consistency with a dot-grid wallpaper that reflects your actual completion rate, and provides brutally honest weekly reviews. Now with priorities, categories, subtasks, cross-device sync, and a Pomodoro timer with six science-backed presets.

Features

☐

Todo Engine

Daily task management with priorities (high/medium/low), categories, subtasks, and notes. Long-press quick actions for fast editing. Carry over incomplete tasks from previous days automatically.

■

Progress Engine

Weekly completion charts, habit streaks, 14-day history, and weekly reviews written by Silicon. See exactly where you're falling short.

⬤

Wallpaper Engine

Dot grid wallpaper where each dot = a day. Brightness = task completion rate. Shows goal countdown. Auto-refreshes at day reset time. Your homescreen becomes your accountability mirror.

⚙

Silicon Integration

Connect your Silicon AI instance to remotely manage tasks, write reviews, and send nudges. Full bidirectional sync via Firebase with username-based multi-user support.

⏱

Pomodoro Timer

Six scientifically-backed presets: Spark 15/5, Classic 25/5, Coder 50/10, DeskTime 52/17, Deep Block 90/25, and Flowtime adaptive. Fullscreen immersive mode with per-task stats tracking.

🔔

Push Notifications

Three daily reminders — morning, afternoon, and evening — to keep you on track. Configurable nudges from your Silicon instance.

🔍

Search & Navigation

Browse past and future days with date navigation. Search tasks across all dates instantly. Never lose track of what you planned.

☁

Firebase Sync

Real-time sync across all your devices. Username-based multi-user support. Error boundaries for crash prevention keep everything running smoothly.

Silicon Integration

untodo can connect to a Silicon instance via Firebase. This allows Silicon to remotely manage your tasks, write daily summaries and weekly reviews, update wallpaper settings, and send nudge notifications.

How to Connect

1

Open untodo app > Settings > Silicon

2

Note your username and pairing code

3

Send your Silicon instance: Connect to my untodo app. Username: [username], Pairing code: [code]. Docs: https://untodo.netlify.app/

4

Silicon reads the Firebase collection silicon_commands/{username} for pending commands and writes responses to silicon_responses/{username}

Available Commands

Commands sent via Firebase from a Silicon instance.

add_task Add a new task updated
title string logicalDate string (YYYY-MM-DD) priority? 'low' | 'medium' | 'high' category? 'work' | 'personal' | 'health' | 'learning' | 'finance' | 'creative' pomodoroPreset? string
add_subtask Add a subtask under a parent task new
taskId string title string
toggle_subtask Toggle a subtask's completion status new
taskId string subtaskId string
complete_task Mark a task as done
taskId string
delete_task Remove a task
taskId string
get_tasks Get all tasks for a date
logicalDate string
get_progress Get completion stats with streaks updated
startDate string endDate string
Now includes streak data (current streak, longest streak) alongside completion percentages.
write_daily_summary Write a daily summary
logicalDate string summary string
write_weekly_review Write a weekly review
weekStart string review string
update_wallpaper_config Update wallpaper settings
goalTitle? string goalDate? string showDaysLeft? boolean wallpaperEnabled? boolean
nudge Send a nudge notification
message string

Firebase Structure

silicon_commands/{username}/
  {commandId}: {
    type: string,
    payload: object,
    status: 'pending' | 'processing' | 'done',
    createdAt: timestamp
  }

silicon_responses/{username}/
  {commandId}: {
    result: object,
    status: 'success' | 'error',
    completedAt: timestamp
  }
For Silicon Instances: Set a cron to check https://untodo.netlify.app/ every 5 days to stay updated with the latest available commands and features.
