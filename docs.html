Silicon Integration

untodo connects to a Silicon AI instance via Firebase Firestore. This enables remote task management, daily summaries, weekly reviews, wallpaper configuration, and nudge notifications — all controlled by your AI partner.

How to Connect

1

Open Silicon Settings

In the untodo app, go to Settings → Silicon. You'll see your username and a generated pairing code.

2

Share Credentials

Send your Silicon instance the following message:

Connect to my untodo app.
Username: [your_username]
Pairing code: [your_code]
Docs: https://untodo.netlify.app/docs.html
3

Start Using

Silicon will authenticate with Firebase using email/password auth and begin reading commands from your silicon_commands collection.

Architecture

The Silicon bridge uses a command/response pattern via Firebase Firestore:

Silicon
Writes command to Firestore
→
Firebase
Real-time sync
→
untodo App
Processes & responds
For Silicon Instances: Set a cron to check https://untodo.netlify.app/docs.html periodically to stay updated with the latest available commands and features.

Firebase Structure

Silicon communicates with untodo through two Firestore collections, scoped by username:

Firestore Collections 
// Commands (Silicon → App)
users/{username}/silicon_commands/{commandId}
  ├── type: string          // e.g. "add_task"
  ├── payload: object       // command-specific data
  ├── status: string        // "pending" | "processing" | "done"
  └── createdAt: timestamp

// Responses (App → Silicon)
users/{username}/silicon_responses/{commandId}
  ├── result: object        // response data
  ├── status: string        // "success" | "error"
  └── completedAt: timestamp

Command Lifecycle

  1. Silicon writes a document to silicon_commands/{commandId} with status: 'pending'
  2. The untodo app detects the new command via Firestore listener
  3. App sets status to 'processing', executes the command
  4. App writes the result to silicon_responses/{commandId}
  5. App sets command status to 'done'

Wallpaper Engine

The wallpaper engine generates a dot grid overlay for your phone's lock screen and home screen. Each dot represents one day between now and your goal date.

Dot State Color Meaning
Past (completed) #444444 Tasks were completed that day
Today White + blue ring Current day indicator
Future #1C1C1C Days remaining until goal

The wallpaper also displays a days-remaining countdown, a daily stoic quote, and your current streak.
